Guidelines for Telepresence Room Booking
Telepresence Booking portal

Link :  http://discovery.bits-pilani.ac.in:8080/reservation/


The Steps for using the Telepresence Booking portal are mentioned below:

General Steps:-
  • First time users can use the Forgot Password Link to generate a password for their account.
  • Login with your BITS Email user id and generated password
  • On welcome page click your option for booking or cancel Booking
Booking Steps:-
  • Select two or more campuses for which Booking has to be done.
  • Select Type of room like Telepresence Conference or Class room.
  • Select the desired date.
  • Click on Get Availability.
  • The returned calendar will show you availability of rooms, you can choose among the Green slots to Book.
  • After fixing schedule, enter basic details about the booking, its title, description etc.
  • Finally fix the schedule by proceeding.
